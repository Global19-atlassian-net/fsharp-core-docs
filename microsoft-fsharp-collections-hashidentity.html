Signature:IEqualityComparer<'T>
Modifiers:

Physical hashing (hash on reference identity of objects, and the contents of value types).
Hash using LanguagePrimitives.PhysicalEquality and LanguagePrimitives.PhysicalHash, That is, for value types use GetHashCode and Object.Equals (if no other optimization available), and for reference types use System.Runtime.CompilerServices.RuntimeHelpers.GetHashCode and reference equality.
